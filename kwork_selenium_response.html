Задача:
Нужно настроить автоматический процесс создания SEO-страниц на Tilda на основе данных из Google Sheets с использованием сервиса n8n (облачная версия).
Что должно быть реализовано:
Интеграция Google Sheets и n8n (уже настроено!)
Настроить чтение данных из конкретного листа (каждая колонка = 1 страница).
Поля: Slug, Title, Description, H1, Text_1, Text_2, ImageURL, ImageAlt.
Создание страниц на Tilda через API:
Для каждой строки/колонки в Google Sheets автоматически создавать страницу в выбранном проекте Tilda.
Страница создаётся на основе шаблонной страницы (ID предоставлю).
Заполнение контента:
Подставлять H1, Text_1, Text_2, ImageURL, ImageAlt в соответствующие блоки.
Настроить SEO-поля (Title, Description, Slug).
Публикация страницы:
После заполнения контента страница должна быть опубликована через page/publish.
Исходные данные:
Tilda API: Public Key, Secret Key, Project ID, Template Page ID — предоставлю.
Google Таблица: готовая структура с данными.
n8n: облачная версия (https://winchester1.app.n8n.cloud/).
Результат:
Настроенный и рабочий workflow в n8n, который:
читает данные из Google Sheets,
создаёт страницы в Tilda,
заполняет контент и SEO,
публикует готовые страницы.
Документация по работе (инструкция как запускать).
Дополнительно:
Подрядчик должен знать Tilda API (https://help.tilda.cc/api), n8n, Google Sheets API.
Желательно — пример подобных проектов.
